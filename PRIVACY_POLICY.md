# TrafficForge Privacy Policy

Developer: Ankur Pathak  
Effective: May 2, 2026

TrafficForge is a Chrome extension for local traffic control and debugging.

## Data Handling

- TrafficForge stores rule configuration locally in the browser using Chrome storage.
- It may capture recent request metadata locally so you can review which requests matched your rules.
- It does not intentionally upload traffic data, captured requests, or rule configuration to any remote server.
- It does not include analytics, advertising, or third-party telemetry.

## Permissions

TrafficForge requests browser permissions only to:

- apply user-defined request blocking and modification rules
- inspect request metadata needed for rule matching
- scope rules to the current tab when selected

## Network Traffic

TrafficForge can interact with browser-visible request traffic for supported protocols and request types. It operates according to the rules configured by the user.

## Third-Party Services

TrafficForge does not rely on third-party services for core functionality.
